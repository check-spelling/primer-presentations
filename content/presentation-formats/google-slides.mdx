The GitHub presentation template is available for use in the Google Slides template gallery.

## How to use

### From the Template Gallery

In Google Slides, start a presentation with the template by selecting it from the Template Gallery.

<img
  width="656"
  alt="Select the GitHub presentation template to get started"
  src="https://user-images.githubusercontent.com/10384315/101223163-3f230a80-3640-11eb-8623-4f6d9d5cf926.png"
/>

If you do **not** see the template, expand the **Template Gallery** by clicking on the **Template Gallery ‚ÜïÔ∏è** button in the top right of the gallery area. This will expand and all available templates for GitHub will be available.

<img
  width="656"
  alt="The Template Gallery ‚ÜïÔ∏è button in the top right of the gallery area"
  src="https://user-images.githubusercontent.com/10384315/101223398-d25c4000-3640-11eb-8d88-99f503379b7d.png"
/>

#### Help! I can't see the template gallery.

If the template gallery does not appear on your Google Slides homescreen follow the steps below to make it visible.

1. Open the menu by clicking on the **hamburger menu** (button with three stacked lines) in the top left of the screen.

   <img
     width="656"
     alt="Open the menu in the top left of the screen"
     src="https://user-images.githubusercontent.com/10384315/101223162-3f230a80-3640-11eb-93af-f73468751306.png"
   />

2. Click **Settings**

   <img
     width="279"
     alt="Click 'Settings'"
     src="https://user-images.githubusercontent.com/10384315/101223160-3f230a80-3640-11eb-9907-35582a4ed37a.png"
   />

3. In the **Templates** section, check the box next to **Display recent templates on homescreen**.

   <img
     width="656"
     alt="From settings, check the box Display recent templates on homescreen"
     src="https://user-images.githubusercontent.com/10384315/101223158-3df1dd80-3640-11eb-8348-65ff9695e214.png"
   />

4. Click **OK**. You should now see the **Template Gallery** on your homescreen.

### From a blank presentation

If you have already created a presentation file in Google Slides, the GitHub presentation template can still be applied using the "Theme" tab.

Click the **Theme** button to open the theme chooser.

<img
  width="504"
  alt="Screenshot of the theme chooser in Google Slides"
  src="https://user-images.githubusercontent.com/10384315/101224016-4d722600-3642-11eb-9bc5-f309bf17c2d1.png"
/>

In the Themes sidebar that opens on the right side of the screen, click **Import theme** at the bottom

<img
  width="297"
  alt="Click import theme"
  src="https://user-images.githubusercontent.com/10384315/101224053-74c8f300-3642-11eb-97e0-002407b31400.png"
/>

A pop up modal will appear. In the search box type, "GitHub presentation template" and then select the presentation file to import the theme.

<img
  width="656"
  alt="Search for the GitHub presentation template to select the theme"
  src="https://user-images.githubusercontent.com/10384315/101223918-07b55d80-3642-11eb-8d74-c1f221237362.png"
/>
